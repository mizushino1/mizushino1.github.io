git## [Dota 2 Player & Hero Tracker](https://mizushino1.github.io/midterms/dota2/)

A comprehensive web application designed to track player performance and explore hero data using real-time information. This project was developed as a midterm requirement for a **Web Development** course, focusing on API integration, responsive design, and dynamic data rendering.

---

### 🚀 Features

*   **Player Search:** Look up specific players using their unique **Dota 2 Account ID**.
*   **Match History:** Displays a detailed list of recent matches, including win/loss status and performance metrics.
*   **Hero Statistics:** View comprehensive player stats for individual heroes to see mastery levels and playstyles.
*   **Hero Encyclopedia:** A complete list of all Dota 2 heroes with a built-in search/filter function.
*   **Hero Details:** Deep dive into specific heroes to view their core attributes, base stats, and unique abilities.

---

### 🛠️ Tech Stack

*   **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+)
*   **Styling:** Bootstrap
*   **API:** [OpenDota API](https://docs.opendota.com/) for real-time player and hero data.
*   **Design:** UI components inspired by the Dota 2 game interface.

---

### 📖 How to Use

1.  **Search Players:** Navigate to the Search bar and input a valid Dota 2 Account ID (e.g., `76482434`).
2.  **Explore Heroes:** Click on the "Heroes" tab to browse the roster. Clicking a hero card will open their specific stats and skills page.
3.  **View Results:** Data is fetched dynamically, providing the most up-to-date information available from the Steam network.
